# AI 开发学习笔记

这个仓库记录了我学习 AI 开发的过程。

## 已完成

- Python 基础练习（变量、循环、判断、列表）
- 大模型 API 调用（DeepSeek）
- 信息提取 JSON 输出

## 如何运行

1. 安装依赖：`pip install openai`
2. 获取 DeepSeek API Key
3. 运行任意 `.py` 文件

## 文件说明

- `hello.py` - 第一个程序
- `score.py` - 成绩等级判断
- `sum.py` - for 循环求和
- `guess.py` - 猜数字游戏
- `json_extract.py` - 调用 API 提取信息
